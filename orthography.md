Orthography
===

Although the standard Latin alphabet has been used throughout this website, the original doc, and mostly everywhere, other systems/scripts have been created for style, speed, or just fun.

Note: The sentence shown in each example is "***bolszekisumantuxi tje, ju ije haja za alo***".

Punctuation, Capitalization and formatting
---
There are 2 styles of LRI, `eczejo` and `kirosejo` (small way and big way). 

### Eczejo
In eczejo, a linebreak follows the end of each sentence and only proper roots are capitalized. `tje` is always follwed by a commma and a newline unless the context very small. Quotations are formed by a `:` followed by `"<the text, in quotes>".`. If a compound has been used before, it is only nessicary to write the head of the compound, with the first letter capitalized to refer back to it. This can also be dome with names, if `kunsi-sa` is the only `sa` in the writing so far, `Sa` can be written instead. Blank lines can be used to seperate "paragraphs". This system is mainly a romanization for [vimsonjo](/orthography/vimsonjo.md).

```
kjo ju eczejo.
nidu vimu namo: "tje" tje,
paju kjo.
ix vimu kep lanRepunido.
kjo tje, ix kjenlanu Ido.
nami kja kirosu tje,
nidu linjo nova.
et lanu: "kjo ju laneczo".

```
