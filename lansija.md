Sign Language
===

History
---
Kunsi-sa had this idea shortly after the creation of vimsonjo and has thought of doing it many times. After creating the 22 radical system for kijazu Kunci, he decided to create it. 

Operation
---
Each radical in vimsonjo has a corresponding sign made with the arms and hands. Some common words have there own signs.

List of Radicals
---

| Name | Sign |
| ---- | ---- |
| jan  | Arms in a triangle, with the fingertips overlaping at the base of the neck |
| luka | _Jan_, but with one arm sticking out and parallel to the other |
| lawa | Hands forming a circle infront of the face |
| en   | Arms forming a "plus" sign |
| ala  | Arms froming a "x" |
| uta  | _Jan_, but with the fingertips at the mouth |
| kama | Arms in a triangle, fingers facing upwards |
| linja | Arms infront of the chest, with each palm on the forearm of the opposite arm |
